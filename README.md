# Engram Hopfield Model

Sparse attractor network model exploring fear and extinction engrams in lateral amygdala.

Current features:
- Sparse Hopfield network
- Asynchronous updates
- Cue-driven retrieval
- Parameter sweeps (cue fraction, beta, theta)

Planned extensions:
- Multiple stored memories (fear + extinction)
- PV-mediated inhibition
- Competitive allocation mechanisms
